# My Interactive Virtual CV & Portfolio

An elegant, responsive, and interactive single-page portfolio showcasing my academic milestones, student governance leadership, and technical research aspirations.

## 🚀 Live Demo
View the live website here: [https://48719544.github.io/virtual-cv/](https://48719544.github.io/virtual-cv/)

## 🛠️ Features & Elements
- **Interactive Slideshow Gallery:** A fully automated, interactive slideshow displaying key milestones in campus leadership, technical hackathons, and public speaking. Slides automatically transition every 4 seconds or pause/reset when manually clicked.
- **Dynamic Light/Dark Theme:** A toggle switch allowing visitors to seamlessly switch between dark and light modes.
- **Embedded Research Proposal:** Features a high-visibility module presenting my proposed Honours research topic (*A Comparative Evaluation of Unsupervised Anomaly Detection Algorithms for Credit Card Fraud Detection*), complete with direct links to view or download the full `RESEARCH_PROPOSAL.pdf`.
- **Integrated Geo-Location:** A clickable marker for GA MASEMOLA that redirects visitors straight to Google Maps.

## 💻 Web Technologies Used
- **HTML5:** Semantic structuring.
- **CSS3:** Responsive layout grids, modern custom properties (variables), transitions, and responsive mobile overrides.
- **JavaScript (Vanilla):** Light/Dark theme controller, interactive carousel control, dynamic autoplay timer logic, and map integration.
